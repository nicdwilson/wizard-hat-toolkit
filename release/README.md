# Wizard Hat Toolkit v1.5.0 - Release Package

## 🎉 What's New

This release introduces the **WooCommerce Blueprint Importer**, a powerful new feature that allows you to quickly configure WooCommerce stores with predefined setups.

## 📦 Package Contents

- `wizard-hat-toolkit-1.5.0.tgz` - The main addon package
- `README.md` - Main documentation
- `BLUEPRINT_IMPORTER.md` - Detailed blueprint importer guide
- `RELEASE_NOTES.md` - Complete release notes
- `INSTALLATION.md` - Installation and setup guide
- `sample-blueprint.json` - Example blueprint file
- `LICENSE` - GPL license

## 🚀 Quick Start

1. **Install**: Use "Install from disk" in Local with the `.tgz` file
2. **Configure**: Set up your GitHub token for premium plugin access
3. **Import**: Use the new Blueprint Importer tab to upload blueprints

## ✨ Key Features

### Blueprint Importer (New!)
- Upload and validate JSON blueprint files
- Preview imports before applying changes
- Handle conflicts with existing installations
- Configure import settings and preferences
- Track import progress with error reporting

### Existing Features
- Plugin Management (WordPress.org, GitHub, Premium)
- Theme Management (WordPress.org, GitHub, Premium)
- Site Configuration (Country/locale switching)
- Plugin Updates (Automated marketplace updates)

## 📋 System Requirements

- **Local** 5.x or higher
- **GitHub account** (for premium plugin access)
- **Node.js** (for building from source)
